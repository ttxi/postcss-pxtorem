# umi project

## Getting Started

Install dependencies,

```bash
yarn
```

Start the dev server,

```bash
yarn start
```

## 合法的提交日志格式如下(emoji 和 模块可选填)

`[<emoji>] [revert: ?]<type>[(scope)?]: <message>`

- 💥 feat(模块): 添加了个很棒的功能
- 🐛 fix(模块): 修复了一些 bug
- 📝 docs(模块): 更新了一下文档
- 🌷 UI(模块): 修改了一下样式
- 🏰 chore(模块): 对脚手架做了些更改
- 🌐 locale(模块): 为国际化做了微小的贡献
